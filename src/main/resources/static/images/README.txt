# Instruções para adicionar o logo

1. Salve sua imagem do logo (o ícone circular com as barras) como:
   - `logo.png` (nesta pasta) - para o cabeçalho do site
   - `favicon.png` (nesta pasta) - para o ícone da aba do navegador

2. Recomendações de tamanho:
   - logo.png: 200x200 pixels ou maior
   - favicon.png: 32x32 pixels ou 64x64 pixels

3. Após salvar as imagens, recarregue a página no navegador (Ctrl+F5 para forçar reload)

4. O logo aparecerá automaticamente no cabeçalho e como favicon na aba!
